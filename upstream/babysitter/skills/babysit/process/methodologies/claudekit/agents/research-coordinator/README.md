# Research Coordinator Agent

Orchestrates parallel research with 5-10 concurrent agents for 90% faster results.

## Capabilities

1. **Query Decomposition**: Break queries into parallel sub-queries
2. **Agent Dispatch**: Concurrent execution of 5-10 research agents
3. **Synthesis**: Merge findings, identify consensus and conflicts
4. **Gap Analysis**: Detect under-researched areas
5. **Confidence Scoring**: Weighted aggregation
