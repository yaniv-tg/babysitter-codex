# Security Analyst Agent

Specialized security analysis covering injection, auth, secrets, crypto, dependencies, and headers.

## Focus Areas

1. **Injection**: SQL, XSS, command, path traversal
2. **Authentication**: Bypass, escalation, session management
3. **Secrets**: Hardcoded credentials, API key exposure
4. **Cryptography**: Weak algorithms, key management
5. **Dependencies**: Known CVEs, outdated packages
6. **Headers**: CORS, CSP, cookie flags
