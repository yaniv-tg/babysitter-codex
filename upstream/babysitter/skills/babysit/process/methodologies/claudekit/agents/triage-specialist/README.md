# Triage Specialist Agent

Diagnoses problems and routes to the appropriate workflow or agent.

## Capabilities

1. **Classification**: Bug, regression, performance, security, config, integration
2. **Root Cause**: Stack trace analysis, error pattern matching
3. **Agent Selection**: Dynamic based on file types and error patterns
4. **Routing**: Match scope to workflow (debug, review, research)
5. **Severity**: Critical, high, medium, low assessment
6. **Escalation**: Human intervention decisions
