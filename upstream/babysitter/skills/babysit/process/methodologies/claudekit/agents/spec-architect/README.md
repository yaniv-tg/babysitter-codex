# Spec Architect Agent

Creates comprehensive feature specifications from codebase research.

## Specification Components

1. **Scope**: Feature boundaries and non-goals
2. **Requirements**: Functional and non-functional
3. **Acceptance Criteria**: Testable, measurable
4. **Architecture**: Decision records with rationale
5. **Implementation Plan**: Phased, dependency-ordered
6. **Risk Analysis**: Identification and mitigation
7. **API Contracts**: Data models and interfaces
8. **Test Strategy**: Unit, integration, E2E mapping

## Invocation

Used by process: `methodologies/claudekit/claudekit-spec-workflow`
