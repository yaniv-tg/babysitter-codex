# Performance Analyst Agent

Identifies performance bottlenecks and optimization opportunities in code changes.

## Focus Areas

1. **Algorithms**: Complexity analysis, Big-O flagging
2. **Resources**: Memory leaks, handle cleanup, timer leaks
3. **Database**: N+1 queries, missing indexes
4. **Caching**: Strategy evaluation, invalidation
5. **Async**: Blocking operations, promise optimization
6. **Bundle**: Size impact, tree-shaking, code splitting

## Invocation

Used by process: `methodologies/claudekit/claudekit-code-review`
