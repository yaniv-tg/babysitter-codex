# Spec Execution Skill

6-phase iterative implementation workflow with quality-gated convergence.

## Phases

1. Implementation, 2. Test Writing, 3. Code Review, 4. Iterative Improvement, 5. Atomic Commit, 6. Progress Tracking

## Quality Gate

Threshold: 80 (configurable). Max 3 improvement cycles before human review.
