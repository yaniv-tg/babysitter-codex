# COG Daily Intelligence Skill

Personalized verified news briefs with strict quality standards.

## Quick Start

```bash
babysitter run:create --process methodologies/cog-second-brain/cog-intelligence-cycle \
  --input '{"vaultPath":"./cog-vault","mode":"daily-brief","userName":"Alex","rolePack":"engineer"}'
```

## Quality Standards

- 7-day freshness on all sources
- 95%+ source accuracy target
- Confidence levels on every claim
- Verification-first: no unsourced intelligence
