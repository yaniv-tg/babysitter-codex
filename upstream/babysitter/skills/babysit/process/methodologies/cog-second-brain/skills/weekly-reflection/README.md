# COG Weekly Reflection Skill

Cross-domain pattern analysis for weekly check-ins.

## Quick Start

```bash
babysitter run:create --process methodologies/cog-second-brain/cog-reflection-synthesis \
  --input '{"vaultPath":"./cog-vault","mode":"weekly-checkin","userName":"Alex","rolePack":"engineer"}'
```

## What Gets Analyzed

- **Personal patterns** - Energy, mood, productivity trends
- **Professional patterns** - Skills, career trajectory, industry alignment
- **Project patterns** - Velocity, blockers, estimation accuracy
- **Cross-domain connections** - How personal affects professional and vice versa
