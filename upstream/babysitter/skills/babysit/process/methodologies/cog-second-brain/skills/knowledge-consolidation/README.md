# COG Knowledge Consolidation Skill

Build structured frameworks from scattered notes.

## Quick Start

```bash
babysitter run:create --process methodologies/cog-second-brain/cog-reflection-synthesis \
  --input '{"vaultPath":"./cog-vault","mode":"knowledge-consolidation","userName":"Alex"}'
```

## Framework Structure

Each consolidated framework includes:
- **Concept Map** - Visual overview of key concepts and relationships
- **Key Principles** - Core ideas distilled from source notes
- **Examples** - Concrete examples from captured experiences
- **Applications** - How to apply the framework in practice
- **Sources** - Attribution to all original notes
