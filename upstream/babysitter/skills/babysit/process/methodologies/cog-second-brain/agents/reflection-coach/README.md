# Reflection Coach Agent

Guides cross-domain pattern analysis and personal/professional strategy evolution.

## Overview

The Reflection Coach facilitates weekly and monthly reflection cycles. It identifies patterns spanning personal, professional, and project domains, and translates them into actionable strategy updates.

## Use Cases

- **Weekly Check-in** - Cross-domain pattern analysis over the past week
- **Monthly Evolution** - Track how themes evolve across weeks
- **Strategy Updates** - Adjust personal and professional goals based on patterns
- **Quality Refinement** - Iteratively improve reflection depth and actionability
- **Growth Tracking** - Monitor personal and professional growth trajectories

## Key Principle

Cross-domain synthesis with strict separation: analyze personal, professional, and project domains independently, then identify connections between them. All insights include confidence levels.
