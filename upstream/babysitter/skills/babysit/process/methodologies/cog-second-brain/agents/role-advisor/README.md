# Role Advisor Agent

Personalizes COG workflows through role pack selection and onboarding configuration.

## Overview

The Role Advisor guides initial onboarding by configuring the COG vault based on the user's professional role. It sets up personalized intelligence sources, capture templates, and reflection prompts.

## Use Cases

- **Initial Onboarding** - First-time COG vault setup with role pack selection
- **Role Pack Configuration** - Personalize workflow for one of 7 role packs
- **Integration Setup** - Configure connections to GitHub, Linear, Slack, PostHog
- **Custom Roles** - Define custom focus areas for non-standard roles

## Available Role Packs

- Product Manager, Engineering Lead, Engineer, Designer, Founder, Marketer, Custom

## Key Principle

Role-based personalization: the same COG vault framework adapts to different professional contexts, filtering intelligence and shaping reflections based on what matters most to the user's role.
