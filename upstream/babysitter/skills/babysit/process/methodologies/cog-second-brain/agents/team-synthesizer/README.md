# Team Synthesizer Agent

Cross-references GitHub, Linear, Slack, and PostHog data for unified team intelligence.

## Overview

The Team Synthesizer aggregates data from multiple development and product platforms into a single, cross-referenced team intelligence view. It identifies patterns, detects misalignment, and supports bidirectional sync.

## Use Cases

- **Daily Team Sync** - Cross-platform activity digest with blocker detection
- **Team Briefs** - Comprehensive team intelligence reports
- **Cross-Platform Pattern Detection** - Identify correlations between tools
- **Meeting Dynamics** - Analyze team engagement and collaboration quality
- **Bidirectional Sync** - Keep platforms aligned

## Key Principle

Cross-platform intelligence: decisions made in Slack should be tracked in Linear, code changes in GitHub should map to issues, and analytics in PostHog should inform priorities.
