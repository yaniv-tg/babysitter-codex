# Intelligence Analyst Agent

Gathers, verifies, and synthesizes news intelligence with strict quality standards.

## Overview

The Intelligence Analyst applies verification-first methodology to all intelligence gathering. Every claim requires a source, all sources must be within 7 days, and the target is 95%+ accuracy.

## Use Cases

- **Daily Briefs** - Personalized verified news filtered by role and interests
- **URL Analysis** - Extract insights from saved URLs with credibility scoring
- **Source Verification** - Cross-reference claims across multiple sources
- **Comprehensive Analysis** - Deep 7-day strategic analysis (~8-12 min)
- **Trend/Risk/Opportunity Analysis** - Pattern detection across accumulated data

## Key Principles

- 7-day freshness requirement on all sources
- 95%+ source accuracy target
- Confidence levels on all claims
- Verification-first: no unsourced intelligence
