# Framework Builder Agent

Consolidates scattered notes into structured knowledge frameworks.

## Overview

The Framework Builder transforms accumulated knowledge from daily captures, weekly reflections, and monthly analyses into permanent, structured frameworks stored in 05-knowledge.

## Use Cases

- **Framework Building** - Construct structured frameworks from themed note clusters
- **Monthly Consolidation** - Transform a month of scattered notes into permanent knowledge
- **Framework Refinement** - Iteratively improve framework quality and depth
- **Knowledge Gap Analysis** - Identify areas needing additional capture
- **Cross-Framework Linking** - Maintain connections between related frameworks

## Key Principle

Source attribution preservation: all original sources from captured notes are maintained through the consolidation process. Frameworks always trace back to their source material.
