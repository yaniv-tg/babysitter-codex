# Knowledge Curator Agent

Classifies, organizes, and curates content across the COG vault with strict domain separation.

## Overview

The Knowledge Curator is the gatekeeper for all content entering the vault. It classifies raw thoughts, extracts references, enforces domain separation, and assesses content quality.

## Use Cases

- **Braindump Classification** - Classify raw thoughts into personal, professional, and project domains
- **Quality Assessment** - Score captured content against quality thresholds
- **Note Discovery** - Identify scattered notes ready for framework consolidation
- **Theme Clustering** - Group related notes for knowledge synthesis
- **Content Health** - Monitor overall vault content quality

## Key Principle

Strict domain separation: personal content (02-personal) and professional content (03-professional) are never mixed. All classifications include confidence levels.
