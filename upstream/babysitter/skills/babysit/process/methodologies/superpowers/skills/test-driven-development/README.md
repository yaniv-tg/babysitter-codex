# Test-Driven Development Skill

RED-GREEN-REFACTOR cycle adapted from [Superpowers Extended](https://github.com/pcvelz/superpowers).

## Purpose

Enforce disciplined TDD: write failing test first, verify failure, write minimal code, verify pass, refactor, commit.

## Integration

- **Used by:** All implementation processes
- **Process file:** `../../test-driven-development.js`
