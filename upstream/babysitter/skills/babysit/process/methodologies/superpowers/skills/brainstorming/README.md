# Brainstorming Skill

Socratic design refinement adapted from the [Superpowers Extended](https://github.com/pcvelz/superpowers) methodology.

## Purpose

Turn rough ideas into fully formed designs through natural collaborative dialogue before any code is written.

## Process Flow

1. Explore project context (files, docs, commits)
2. Ask clarifying questions one at a time
3. Propose 2-3 approaches with tradeoffs
4. Present design in sections for incremental approval
5. Write design document
6. Hand off to writing-plans

## Integration

- **Input from:** User request or project kickoff
- **Output to:** `writing-plans` skill
- **Process file:** `../../brainstorming.js`
