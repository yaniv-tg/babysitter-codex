# Task Decomposition Skill

Plan-to-tasks conversion adapted from [Spec Kit](https://github.com/github/spec-kit) by GitHub.

## Purpose

Convert technical plans into actionable development tasks with dependency graphs and execution ordering.

## Integration

- **Input from:** `planning-design` skill with approved technical plan
- **Output to:** `cross-artifact-analysis` skill and then `implementation-execution` skill
- **Process file:** `../../spec-kit-planning.js`
