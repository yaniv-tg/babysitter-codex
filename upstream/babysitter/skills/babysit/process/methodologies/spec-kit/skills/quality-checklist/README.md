# Quality Checklist Skill

Quality validation and checklist adapted from [Spec Kit](https://github.com/github/spec-kit) by GitHub.

## Purpose

Validate implementation quality through multi-dimensional scoring and remediation loops.

## Integration

- **Input from:** `implementation-execution` skill with completed artifacts
- **Output to:** Final results or back to `implementation-execution` for remediation
- **Process file:** `../../spec-kit-implementation.js`
