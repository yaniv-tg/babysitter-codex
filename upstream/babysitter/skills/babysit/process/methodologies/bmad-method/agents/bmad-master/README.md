# BMad Master

The BMad Master agent is the central orchestrator of the BMAD Method framework. It coordinates workflows across all specialized agents and ensures quality across lifecycle phases.

## Quick Reference

- **Agent ID:** `bmad-master`
- **Persona:** Master Task Executor
- **Primary Focus:** Orchestration, validation, consensus
- **Processes:** bmad-orchestrator, bmad-party-mode

## When to Use

Use BMad Master when you need cross-agent coordination, artifact validation, or multi-perspective consensus synthesis. It serves as the glue between specialized agents.
