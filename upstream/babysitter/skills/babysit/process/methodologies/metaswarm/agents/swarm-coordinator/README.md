# Swarm Coordinator Agent

Meta-orchestrator adapted from [Metaswarm](https://github.com/dsifry/metaswarm) by David Sifry.

## Role

Manages parallel issues across worktrees, spawns Issue Orchestrators, detects conflicts, and balances workload.

## Used By

- `metaswarm-swarm-coordinator` process
- `external-tool-coordination` skill
