# Designer Agent

UX/API design reviewer adapted from [Metaswarm](https://github.com/dsifry/metaswarm) by David Sifry.

## Role

Reviews UX/API design and developer experience as one of the 6 mandatory design review gate approvers.

## Used By

- `metaswarm-design-review` process (parallel reviewer)
- `design-review-gate` skill
