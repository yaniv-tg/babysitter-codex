# Security Design Agent

Security design reviewer adapted from [Metaswarm](https://github.com/dsifry/metaswarm) by David Sifry.

## Role

Conducts threat modeling and OWASP Top 10 analysis as one of the 6 mandatory design review gate approvers.

## Used By

- `metaswarm-design-review` process (parallel reviewer)
- `design-review-gate` skill
