# Persistent Memory Skill

## Overview

Cross-session knowledge persistence for decisions, discoveries, and bugfix patterns. Searchable by tags, type, and relevance.

## Observation Types

| Type | Content | Example |
|------|---------|---------|
| **Decision** | Choices with rationale | "Use RTK over raw Redux" |
| **Discovery** | Codebase insights | "API routes use camelCase" |
| **Bugfix** | Resolution patterns | "Race condition fix: add mutex" |
