# Codebase Sync Skill

## Overview

Convention discovery and rule generation from codebase analysis. Implements the /sync command from Pilot Shell.

## Capabilities

| Capability | Description |
|-----------|-------------|
| **Exploration** | Project structure scanning, language/framework detection |
| **Indexing** | Semantic search indexes for code, tests, config, APIs |
| **Convention Discovery** | Code style, testing, git, documentation patterns |
| **Rule Generation** | Convert conventions to enforceable rules |

## Attribution

Adapted from the /sync command in [Pilot Shell](https://github.com/maxritter/pilot-shell) by Max Ritter.
