# Spec-Driven Development Skill

## Overview

Provides the methodology for creating complete, reviewable specifications through semantic search, clarifying questions, and structured generation.

## Capabilities

| Capability | Description |
|-----------|-------------|
| **Semantic Search** | Codebase search to inform spec context |
| **Question Resolution** | Ambiguity identification and resolution |
| **Spec Generation** | Structured spec with goals, tasks, criteria |
| **Iterative Refinement** | Feedback-driven spec revision |
