# Session Memory Skill

Mandatory memory persistence across session resets. Three markdown surfaces in `.claude/cc10x/` provide continuity for all CC10X workflows.

## Files

- `activeContext.md` -- focus, decisions, learnings
- `patterns.md` -- conventions, gotchas, architecture
- `progress.md` -- completed tasks, verification evidence

## Iron Law

Every workflow: LOAD at start, UPDATE at end.

## Invocation

Managed by: `methodologies/cc10x/cc10x-router` (load-memory and update-memory tasks)
