# GitHub Researcher Agent

External research agent that searches GitHub repositories and packages to discover existing solutions, patterns, and best practices.

## Research Outputs

- Relevant repositories with links
- Package recommendations
- Best practices and patterns
- Known pitfalls and anti-patterns
- Confidence-rated findings

## Invocation

Used by processes: `methodologies/cc10x/cc10x-plan`, `cc10x-debug` (indirectly via planner/bug-investigator)
