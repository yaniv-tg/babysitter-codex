# Planner Agent

Comprehensive planning agent used in the PLAN workflow. Researches, brainstorms, and creates actionable plans that flow to BUILD via docs/plans/.

## Planning Process

1. External research and pattern discovery
2. Brainstorm 3+ alternative approaches with trade-offs
3. Create phased plan with milestones and dependencies
4. Include risk assessment and TDD strategy
5. Save to docs/plans/ for build continuity

## Invocation

Used by process: `methodologies/cc10x/cc10x-plan`
