# Mayor Agent

Global coordinator agent adapted from [Gas Town](https://github.com/steveyegge/gastown) by Steve Yegge.

## Role

The Mayor initiates Convoys, distributes work, monitors execution, and handles escalations across the entire Gas Town operation.

## Used By

- `gastown-orchestrator` process (primary orchestrator)
- `gastown-convoy` process (convoy creation)
- `agent-coordination` skill
