# Crew Lead Agent

Persistent collaborator agent adapted from [Gas Town](https://github.com/steveyegge/gastown) by Steve Yegge.

## Role

Long-lived agent maintaining deep context across multiple beads for complex, persistent work.

## Used By

- `gastown-convoy` process (persistent bead execution)
- `gastown-orchestrator` process (complex work assignment)
- `agent-coordination` skill
