# Brainstorming Skill

Requirement clarification adapted from [RPIKit](https://github.com/bostonaholic/rpikit) by Matthew Boston.

## Purpose

Clarify vague requirements through structured questioning before research begins.

## Integration

- **Input from:** User with unclear requirements
- **Output to:** `codebase-research` skill with clarified requirements
- **Process file:** `../../rpikit-brainstorm.js`
