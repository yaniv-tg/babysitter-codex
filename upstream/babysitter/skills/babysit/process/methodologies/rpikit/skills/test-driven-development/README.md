# Test-Driven Development Skill

Test-first practice adapted from [RPIKit](https://github.com/bostonaholic/rpikit) by Matthew Boston.

## Purpose

Ensure all code changes have tests written before implementation.

## Integration

- **Input from:** Plan writing (test strategy) and implementation (step execution)
- **Output to:** Verified test suites as part of implementation
- **Process files:** `../../rpikit-plan.js`, `../../rpikit-implement.js`
