# Plan Writing Skill

Actionable implementation planning adapted from [RPIKit](https://github.com/bostonaholic/rpikit) by Matthew Boston.

## Purpose

Transform research findings into granular, verifiable implementation plans with stakes-appropriate rigor.

## Process Flow

1. Load research findings
2. Classify stakes (low/medium/high)
3. Define success criteria
4. Decompose tasks with test-first strategy
5. Assess risks and rollback
6. Write plan document with approval gate

## Integration

- **Input from:** `codebase-research` skill (research document)
- **Output to:** `plan-implementation` skill for execution
- **Process file:** `../../rpikit-plan.js`
