# Systematic Debugging Skill

Hypothesis-driven debugging adapted from [RPIKit](https://github.com/bostonaholic/rpikit) by Matthew Boston.

## Purpose

Structured defect investigation using hypothesis-driven methodology rather than trial-and-error.

## Integration

- **Input from:** Implementation failure or bug report
- **Output to:** Targeted fix with verification
- **Process file:** `../../rpikit-implement.js` (failure handling)
