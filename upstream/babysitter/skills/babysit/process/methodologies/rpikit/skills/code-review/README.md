# Code Review Skill

Structured code quality assessment adapted from [RPIKit](https://github.com/bostonaholic/rpikit) by Matthew Boston.

## Purpose

Evaluate code changes for quality, design, correctness, and maintainability with scaled depth and soft-gating verdicts.

## Integration

- **Input from:** `plan-implementation` skill or manual invocation
- **Output to:** Security review and/or merge
- **Process file:** `../../rpikit-review.js`
