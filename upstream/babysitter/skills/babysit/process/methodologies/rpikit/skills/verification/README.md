# Verification Skill

Verification-before-completion discipline adapted from [RPIKit](https://github.com/bostonaholic/rpikit) by Matthew Boston.

## Purpose

Ensure all success criteria, tests, and reviews complete before declaring work done.

## Integration

- **Input from:** Step execution and phase checkpoints
- **Output to:** Completion confirmation or failure investigation
- **Process file:** `../../rpikit-implement.js`
