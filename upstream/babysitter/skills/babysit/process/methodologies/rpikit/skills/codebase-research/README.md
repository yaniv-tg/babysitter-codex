# Codebase Research Skill

Systematic codebase exploration adapted from [RPIKit](https://github.com/bostonaholic/rpikit) by Matthew Boston.

## Purpose

Understand existing code before planning or implementing changes. Enforces the Iron Law: understand the problem first.

## Process Flow

1. Clarify request (no code reading)
2. Explore codebase systematically
3. Document findings
4. Transition to next phase

## Integration

- **Input from:** User request or brainstorm output
- **Output to:** `plan-writing` skill for implementation planning
- **Process file:** `../../rpikit-research.js`
