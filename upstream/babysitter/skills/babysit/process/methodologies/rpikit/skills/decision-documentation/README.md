# Decision Documentation Skill

Architecture Decision Records adapted from [RPIKit](https://github.com/bostonaholic/rpikit) by Matthew Boston.

## Purpose

Document significant technical decisions with context, options, and consequences for future reference.

## Integration

- **Input from:** Any point in the RPI workflow
- **Output to:** `docs/decisions/` directory with sequential ADR numbering
- **Process file:** `../../rpikit-decision.js`
