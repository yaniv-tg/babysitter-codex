# Code Reviewer Agent

Code quality assessor adapted from [RPIKit](https://github.com/bostonaholic/rpikit) by Matthew Boston.

## Role

Structured code review with Conventional Comments format and soft-gating verdicts.

## Used By

- `rpikit-review` process (code review phase)
- `rpikit-implement` process (final code review)
- `code-review` skill
