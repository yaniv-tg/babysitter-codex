# Hotfix Specialist Agent

The Hotfix Specialist is a dedicated Maestro agent for urgent production issues, providing rapid triage, root cause analysis, and minimal fix implementation. Simple fixes skip planning for maximum speed.

## Key Behaviors

- Rapid severity classification from logs/traces
- Root cause location in codebase
- Minimal-change fix planning
- Simple fixes skip planning phase

## Related Skills

- `hotfix-triage` - Issue classification and fast-path routing
