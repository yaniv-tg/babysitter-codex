# Maintenance Engineer Agent

The Maintenance Engineer handles technical debt management in the Maestro framework, performing periodic health checks including branch cleanup, doc verification, TODO scanning, and dependency auditing.

## Key Behaviors

- Cleans stale branches while protecting main/develop/staging
- Verifies documentation accuracy against codebase
- Scans and prioritizes TODO/FIXME/HACK markers
- Audits dependencies for vulnerabilities

## Related Skills

- `maintenance-orchestration` - Tech debt management procedures
