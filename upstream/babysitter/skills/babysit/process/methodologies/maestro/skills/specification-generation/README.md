# Specification Generation Skill

Converts requirements into structured technical specifications. Operates in a feedback loop between PM and Architect until the spec is approved.

## Inputs
- Interview results or requirements document
- Project context
- Architect feedback history

## Outputs
- Technical specification
- Architecture design
- Technology stack with rationale
- Data model and API contracts
- ADRs
