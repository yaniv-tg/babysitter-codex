# Maintenance Orchestration Skill

Technical debt management procedures triggered periodically in the Maestro lifecycle. Covers branch cleanup, documentation, TODOs, coverage, and dependencies.

## Focus Areas
- Branch cleanup (stale, merged, orphaned)
- Knowledge graph sync
- Documentation accuracy verification
- TODO/FIXME/HACK scanning
- Test coverage gap analysis
- Dependency vulnerability audit
