# Story Decomposition Skill

Breaks technical specifications into small, implementable stories with dependency ordering for parallel coder dispatch.

## Inputs
- Technical specification
- Requirements specification
- Maximum story size constraint

## Outputs
- Ordered story list
- Dependency graph
- Story queue for dispatch
- Effort estimates
