# Test Enforcement Skill

Automated test validation and coverage checking with aggressive defaults. Implements Maestro's "turn checks up to 11" philosophy.

## Enforcement Areas
- Unit test execution and coverage
- Integration test execution
- Lint compliance
- Format compliance
- Coverage gap identification
- Test suggestion generation
