# Code Review Gate Skill

Architect-level code review enforcing DRY, YAGNI, proper abstraction, and test coverage. Provides quality scores and specific file:line feedback.

## Principles Enforced
- **DRY** - No unnecessary duplication
- **YAGNI** - No speculative features
- **Proper Abstraction** - Correct encapsulation and separation
- **Test Coverage** - Adequate automated tests

## Inputs
- Implementation files and tests
- Story definition
- Technical specification
- Quality threshold

## Outputs
- Approved/rejected decision
- Quality score
- Violation list with file:line references
- Actionable feedback
