# Hotfix Triage Skill

Urgent issue classification and fast-path routing for production hotfixes. Classifies severity, locates root cause, and determines whether planning can be skipped.

## Severity Levels
- **Critical** - Immediate fix, risk acceptance allowed
- **High** - Same-day fix, standard review
- **Medium** - Next-sprint fix, full planning
