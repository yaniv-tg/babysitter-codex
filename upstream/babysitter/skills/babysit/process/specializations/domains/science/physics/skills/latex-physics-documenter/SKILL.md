# LaTeX Physics Documenter

## Purpose

Provides expert guidance on LaTeX for physics paper preparation, including equation rendering and journal-specific formatting.

## Capabilities

- REVTeX/APS formatting
- Equation and figure handling
- BibTeX reference management
- Tikz/PGF diagram generation
- arXiv submission preparation
- Supplementary material packaging

## Usage Guidelines

1. **Document Setup**: Configure REVTeX for target journal
2. **Equations**: Format complex equations properly
3. **Figures**: Include and format publication-quality figures
4. **References**: Manage bibliography with BibTeX
5. **Submission**: Prepare arXiv and journal submissions

## Tools/Libraries

- LaTeX
- REVTeX
- TikZ
- BibTeX
