# Music Album Creation - Processes Backlog

Future processes and enhancements for the music-album-creation specialization.

## Planned Processes

### Single Song Creation (Standalone)
- **ID**: `single-song-creation`
- **Priority**: High
- **Description**: Streamlined process for creating a single song without album context
- **Inputs**: Song idea, genre direction, mood, optional persona
- **Outputs**: Lyrics, style spec, cover prompt
- **Status**: Could be extracted from song-composition.js with simplifications

### EP Production
- **ID**: `ep-production`
- **Priority**: Medium
- **Description**: 4-6 track mini-album production
- **Use Case**: Shorter format for concept exploration
- **Notes**: Adapt full-album-production with reduced scope

### Artist Evolution Process
- **ID**: `artist-evolution`
- **Priority**: Medium
- **Description**: Evolve an existing persona to a new era/phase
- **Inputs**: Existing persona, evolution direction
- **Outputs**: Updated persona with new era details
- **Notes**: For artists spanning multiple albums

### Remix/Reinterpretation
- **ID**: `song-remix`
- **Priority**: Low
- **Description**: Create alternate versions of existing songs
- **Inputs**: Original song, new genre/style direction
- **Outputs**: New style spec for same lyrics, or reimagined lyrics

### Live Album Adaptation
- **ID**: `live-album-adaptation`
- **Priority**: Low
- **Description**: Adapt studio tracks for live performance versions
- **Inputs**: Studio tracks, live performance aesthetic
- **Outputs**: Live arrangement notes, extended versions

### Soundtrack/Score Composition
- **ID**: `soundtrack-composition`
- **Priority**: Medium
- **Description**: Create cohesive soundtrack for film/game/story
- **Inputs**: Narrative source, scenes, moods
- **Outputs**: Thematic songs, instrumental pieces, leitmotifs

### Collaboration Process
- **ID**: `artist-collaboration`
- **Priority**: Low
- **Description**: Generate content for two artists collaborating
- **Inputs**: Two personas, collaboration concept
- **Outputs**: Blended style, collaborative lyrics

### Genre Exploration Pack
- **ID**: `genre-exploration`
- **Priority**: Medium
- **Description**: Create sample songs across multiple genres for same artist
- **Use Case**: Exploring new directions for an artist
- **Outputs**: 3-5 different genre interpretations

### Music Video Concept
- **ID**: `music-video-concept`
- **Priority**: Low
- **Description**: Create detailed music video treatment for a song
- **Inputs**: Completed song (lyrics, style)
- **Outputs**: Visual treatment, scene breakdown, shot list

### Merch Design Direction
- **ID**: `merch-design`
- **Priority**: Low
- **Description**: Create merchandise visual direction for artist/album
- **Inputs**: Persona, album concept
- **Outputs**: Merch concepts, slogans, visual guidelines
