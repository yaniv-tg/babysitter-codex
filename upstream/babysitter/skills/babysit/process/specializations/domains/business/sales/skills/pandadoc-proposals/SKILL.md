# PandaDoc Proposals

## Overview

The PandaDoc Proposals skill provides integration with PandaDoc for document automation, including proposal template management, document generation from data, e-signature tracking, and pricing table automation. This skill accelerates the quote-to-close process with professional, branded documents.

## Capabilities

### Template Management
- Create and manage document templates
- Configure dynamic content blocks
- Set up conditional content rules
- Maintain brand consistency

### Document Generation
- Generate documents from CRM data
- Populate dynamic fields automatically
- Create custom pricing tables
- Include personalized content

### E-Signature Tracking
- Monitor document viewing activity
- Track signature progress
- Send automated reminders
- Capture completion timestamps

### Pricing Automation
- Build dynamic pricing tables
- Apply discount rules and approvals
- Calculate totals with taxes/fees
- Support multiple currencies

## Usage

### Proposal Generation
```
Generate a customized proposal document from opportunity data, including pricing and terms specific to the deal.
```

### Signature Monitoring
```
Track proposal viewing and signature status to identify follow-up priorities.
```

### Pricing Configuration
```
Create a pricing table with volume discounts and optional line items for customer selection.
```
