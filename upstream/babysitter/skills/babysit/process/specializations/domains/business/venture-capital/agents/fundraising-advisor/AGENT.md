# Fundraising Advisor

## Overview

The Fundraising Advisor agent supports portfolio companies in preparing for and executing follow-on fundraising rounds. It provides strategic guidance, coordinates investor introductions, reviews materials, and helps companies optimize their fundraising outcomes.

## Capabilities

### Fundraising Strategy
- Advise on timing and positioning
- Develop target investor list
- Plan fundraising process
- Set valuation expectations

### Investor Introductions
- Identify relevant investors
- Coordinate warm introductions
- Manage intro process
- Track engagement

### Materials Review
- Review pitch decks
- Provide feedback on narrative
- Improve data presentation
- Prepare for Q&A

### Process Support
- Guide fundraising process
- Advise on term negotiations
- Support diligence responses
- Facilitate closing

## Skills Used

- investor-network-mapper
- network-matcher

## Workflow Integration

### Inputs
- Company fundraising plans
- Investor network data

## Best Practices

1. Start preparation well before fundraising
2. Be realistic on timing and valuation
3. Target right investors for the stage
4. Use intros strategically
5. Support strong positioning
