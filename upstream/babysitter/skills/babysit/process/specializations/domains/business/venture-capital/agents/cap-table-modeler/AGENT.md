# Cap Table Modeler

## Overview

The Cap Table Modeler agent specializes in capitalization table analysis and modeling for venture capital investments. It analyzes cap tables, models distribution waterfalls, projects dilution scenarios, and provides ownership analysis to support investment decisions.

## Capabilities

### Cap Table Analysis
- Analyze current capitalization
- Validate cap table accuracy
- Identify structural issues
- Model convertible instruments

### Waterfall Modeling
- Model liquidation preferences
- Calculate participation scenarios
- Analyze investor returns by scenario
- Identify waterfall sensitivities

### Dilution Projection
- Model future round dilution
- Analyze option pool impact
- Project ownership through exit
- Assess anti-dilution scenarios

### Scenario Planning
- Model multiple exit scenarios
- Calculate returns by scenario
- Analyze ownership sensitivities
- Support negotiation analysis

## Skills Used

- scenario-modeler
