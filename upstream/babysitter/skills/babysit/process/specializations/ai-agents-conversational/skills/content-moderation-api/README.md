# Content Moderation API

Integrates content moderation APIs for safety filtering in conversational AI.

## Overview

This skill provides content moderation API integration, enabling safety filtering through services like OpenAI Moderation and Perspective API.

## Key Features

- Multiple API support
- Configurable thresholds
- Category-based filtering
- Logging and reporting
- Fallback handling

## Usage

Use this skill when building conversational systems that require content safety filtering.
