# SetFit Few-Shot

Implements SetFit few-shot learning for efficient intent classification with minimal training data.

## Overview

This skill provides SetFit training capabilities, enabling accurate intent classification with as few as 8 examples per class.

## Key Features

- Few-shot learning
- No prompt engineering
- Fast training
- Small model footprint
- Contrastive learning

## Usage

Use this skill when building intent classifiers with limited labeled training data.
