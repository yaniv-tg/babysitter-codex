# CrewAI Setup

Configures CrewAI multi-agent orchestration for collaborative AI systems.

## Overview

This skill provides CrewAI setup capabilities, enabling the creation of multi-agent systems with defined roles, tasks, and collaboration patterns.

## Key Features

- Agent role configuration
- Task definition and assignment
- Multiple process types
- Memory and tool integration
- Hierarchical structures

## Usage

Use this skill when building multi-agent systems that require coordinated collaboration with defined roles and responsibilities.
