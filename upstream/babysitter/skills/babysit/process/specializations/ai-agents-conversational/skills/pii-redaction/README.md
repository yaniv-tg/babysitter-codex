# PII Redaction

Detects and redacts personally identifiable information for privacy compliance.

## Overview

This skill provides PII detection and redaction capabilities, enabling privacy-compliant conversational AI through various anonymization strategies.

## Key Features

- Multiple PII type detection
- Various redaction methods
- Reversible anonymization
- Compliance logging
- Custom pattern support

## Usage

Use this skill when building conversational systems that handle sensitive user data and require privacy compliance.
