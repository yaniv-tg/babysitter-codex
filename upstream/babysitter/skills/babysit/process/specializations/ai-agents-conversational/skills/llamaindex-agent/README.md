# LlamaIndex Agent

Sets up LlamaIndex agents and query engines for RAG-powered applications.

## Overview

This skill provides LlamaIndex agent setup capabilities, enabling the creation of RAG-powered agents with various query engine types and reasoning patterns.

## Key Features

- Multiple agent types
- Various query engines
- Sub-question decomposition
- Memory integration
- Multi-document support

## Usage

Use this skill when building RAG-powered agents using the LlamaIndex framework.
