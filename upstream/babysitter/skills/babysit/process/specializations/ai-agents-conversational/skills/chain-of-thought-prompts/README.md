# Chain-of-Thought Prompts

Designs chain-of-thought and step-by-step reasoning prompts for complex problem solving.

## Overview

This skill provides chain-of-thought prompting capabilities, enabling improved reasoning through structured step-by-step thinking patterns.

## Key Features

- Zero-shot and few-shot CoT
- Self-consistency patterns
- Tree-of-thought reasoning
- ReAct integration
- Reasoning verification

## Usage

Use this skill when building systems that require complex reasoning and step-by-step problem solving.
