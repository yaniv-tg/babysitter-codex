# Multimodal Agent Expert

Builds vision-language agent capabilities for multimodal AI systems.

## Overview

This agent specializes in creating AI agents that can understand and reason about both images and text.

## Key Responsibilities

- Integrate vision-language models
- Design image understanding
- Implement multimodal RAG
- Enable visual reasoning

## When to Use

Engage this agent when building AI systems that need to process and reason about visual content.
