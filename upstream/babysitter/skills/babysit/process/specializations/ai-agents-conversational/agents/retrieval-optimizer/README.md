# Retrieval Optimizer

Tunes retrieval parameters and implements reranking for improved RAG quality.

## Overview

This agent specializes in optimizing retrieval quality through parameter tuning, reranking, and hybrid search strategies.

## Key Responsibilities

- Tune retrieval parameters
- Implement reranking
- Configure hybrid search
- Evaluate retrieval quality

## When to Use

Engage this agent when optimizing RAG retrieval quality or implementing advanced retrieval patterns.
