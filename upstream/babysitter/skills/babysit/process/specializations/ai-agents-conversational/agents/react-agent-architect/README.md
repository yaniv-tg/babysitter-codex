# ReAct Agent Architect

Designs ReAct agent loops with thought-action-observation patterns for effective AI reasoning and action.

## Overview

This agent specializes in architecting ReAct-style agents that combine reasoning with action execution through structured loops.

## Key Responsibilities

- Design ReAct loop structures
- Define tool orchestration patterns
- Implement error recovery strategies
- Optimize agent performance

## When to Use

Engage this agent when designing agents that need to reason about tasks before taking actions, particularly for complex multi-step problem solving.
