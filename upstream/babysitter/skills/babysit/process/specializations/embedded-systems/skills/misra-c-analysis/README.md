# MISRA C Static Analysis Skill

MISRA C compliance checking and static analysis integration.

## Purpose

Provides comprehensive MISRA C compliance support:
- MISRA C:2012 rule checking
- Static analysis tool integration
- Violation categorization and reporting
- Deviation record generation
- Coding standard enforcement

## Key Features

- Full MISRA C:2012 support
- Multiple tool integration (PC-lint, Cppcheck, Coverity)
- Deviation management workflow
- CERT C guideline checking
- Compliance reporting

## Integration

Used by processes requiring code quality:
- MISRA C compliance checking
- Functional safety certification
- Device driver development

## Dependencies

- Static analysis tools (PC-lint, Cppcheck, Coverity, Polyspace)
- MISRA C:2012 guidelines
- Rule configuration files
