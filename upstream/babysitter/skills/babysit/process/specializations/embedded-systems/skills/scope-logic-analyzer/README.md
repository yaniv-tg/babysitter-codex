# Oscilloscope/Logic Analyzer Skill

Test equipment integration for signal analysis.

## Purpose

Provides test equipment integration capabilities:
- Oscilloscope measurement automation
- Logic analyzer capture configuration
- Protocol decoding and analysis
- Timing measurement and verification
- Signal integrity analysis

## Key Features

- Multi-vendor instrument support
- Automated measurements
- Protocol decoding (I2C, SPI, UART, CAN)
- Timing analysis
- FFT and spectrum analysis
- SCPI command automation

## Integration

Used by processes requiring signal analysis:
- Signal integrity testing
- Hardware bring-up validation
- Real-time performance validation

## Dependencies

- Oscilloscope software/drivers
- Logic analyzer software (Saleae, Sigrok)
- VISA/SCPI libraries
