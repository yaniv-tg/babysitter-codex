# Unity/Ceedling Test Skill

Embedded unit testing with Unity framework and CMock.

## Purpose

Provides comprehensive embedded unit testing support:
- Unity test case generation
- CMock mock generation for HAL/drivers
- Ceedling build system configuration
- Test coverage analysis (gcov/lcov)
- Off-target testing setup

## Key Features

- Unity assertion macros and fixtures
- Automatic mock generation
- Hardware abstraction mocking
- Code coverage integration
- CI/CD test automation

## Integration

Used by processes requiring testing:
- Embedded unit testing
- Device driver development
- BSP development

## Dependencies

- Unity test framework
- CMock mock generator
- Ceedling build system
- gcov/lcov for coverage
