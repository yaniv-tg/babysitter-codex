# Ghidra/IDA Reverse Engineering Skill

Deep integration with Ghidra and IDA Pro for binary analysis and reverse engineering.

## Quick Reference

- **Slug**: `ghidra-ida-re`
- **Category**: Reverse Engineering
- **ID**: SK-001

## Key Features

- Ghidra headless analysis automation
- IDA Pro scripting via IDAPython
- Decompilation and disassembly parsing
- Function signature extraction
- Call graph and data flow export
- Custom type definition support

## Primary Use Cases

1. Binary vulnerability analysis
2. Malware static analysis
3. Firmware reverse engineering
4. Protocol extraction from binaries
