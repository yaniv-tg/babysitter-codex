# Legacy Decommissioning Specialist Agent

## Overview

The Legacy Decommissioning Specialist agent safely retires legacy systems. It verifies no active usage, archives data, and completes resource cleanup.

## Quick Start

1. **Verify no usage**
2. **Archive data**
3. **Execute shutdown**
4. **Clean up resources**

## Features

- Usage verification
- Data archival
- Staged shutdown
- Resource cleanup
