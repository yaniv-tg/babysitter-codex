# API Modernization Architect Agent

## Overview

The API Modernization Architect agent designs and executes API modernization strategies. It ensures backward compatibility while guiding consumers through migration.

## Quick Start

### Prerequisites

- Current API inventory
- Consumer knowledge
- Target API design goals

### Basic Usage

1. **Inventory APIs**
2. **Design modern API**
3. **Plan migration**
4. **Support consumers**

## Features

### Modernization Patterns

| Pattern | Use Case | Compatibility |
|---------|----------|---------------|
| Versioning | Major changes | High |
| Facade | Legacy wrap | High |
| Gateway | Routing | High |
| Rewrite | Full modernization | Low |

### Deliverables

- Modern API design
- Migration playbook
- OpenAPI specs
- Consumer guides
