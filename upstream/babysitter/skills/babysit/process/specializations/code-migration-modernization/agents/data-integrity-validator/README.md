# Data Integrity Validator Agent

## Overview

The Data Integrity Validator agent ensures data is accurately migrated. It performs comprehensive verification and generates reconciliation reports for audit purposes.

## Quick Start

1. **Define validation rules**
2. **Run verification checks**
3. **Analyze exceptions**
4. **Generate sign-off report**

## Features

- Row count verification
- Checksum validation
- Statistical sampling
- Referential integrity checks
