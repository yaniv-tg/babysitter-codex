# spectron-test-setup

Set up Spectron tests for legacy Electron applications.

**Note: Spectron is deprecated. Use `playwright-electron-config` for new projects.**

## Quick Start

```javascript
const result = await invokeSkill('spectron-test-setup', {
  projectPath: '/path/to/electron-app'
});
```

## Features

- WebDriverIO integration
- Application lifecycle
- Page objects

## Migration

Consider migrating to Playwright Electron testing.

- `playwright-electron-config`
