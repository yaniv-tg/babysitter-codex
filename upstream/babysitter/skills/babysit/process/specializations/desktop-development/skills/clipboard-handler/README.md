# clipboard-handler

Cross-platform clipboard operations.

## Quick Start

```javascript
const result = await invokeSkill('clipboard-handler', {
  projectPath: '/path/to/project',
  framework: 'electron',
  contentTypes: ['text', 'image', 'html', 'files']
});
```

## Features

- Text, images, HTML, RTF
- File paths
- Clipboard monitoring
- Cross-platform API
