# neutralino-js-config

Configure Neutralino.js for lightweight desktop applications.

## Quick Start

```javascript
const result = await invokeSkill('neutralino-js-config', {
  projectPath: '/path/to/project',
  appName: 'my-app',
  windowMode: 'window'
});
```

## Features

- Lightweight (uses OS webview)
- Small binary size
- Native API access
- Cross-platform
