# Kafka Topic Designer

Skill ID: SK-DEA-007

## Purpose

Designs and optimizes Apache Kafka topics and configurations. Provides expertise in topic architecture, partitioning, and producer/consumer configuration.

## Key Features

- Topic naming conventions
- Partition strategy optimization
- Replication configuration
- Retention policy design
- Schema registry integration
- Security configuration

## Applicable Processes

- Streaming Pipeline
- ETL/ELT Pipeline (CDC)
- Feature Store Setup
