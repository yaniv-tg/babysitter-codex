# Schema Evolution Manager

Skill ID: SK-DEA-011

## Purpose

Manages schema evolution and compatibility across data systems. Ensures safe schema changes with backward and forward compatibility.

## Key Features

- Schema compatibility checking
- Breaking change detection
- Migration script generation
- Version management
- Schema registry operations
- Cross-system synchronization

## Applicable Processes

- Streaming Pipeline
- ETL/ELT Pipeline
- Data Catalog
- Pipeline Migration
