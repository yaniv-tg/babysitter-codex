# Incremental Model Strategy Selector

Skill ID: SK-DEA-019

## Purpose

Selects and configures optimal incremental model strategies. Optimizes data transformation efficiency through proper incremental patterns.

## Key Features

- Strategy selection (append, merge, delete+insert)
- Partition pruning optimization
- Unique key configuration
- Schema change handling
- Lookback window optimization
- Late-arriving data handling

## Applicable Processes

- Incremental Model Setup
- dbt Model Development
- Pipeline Migration
