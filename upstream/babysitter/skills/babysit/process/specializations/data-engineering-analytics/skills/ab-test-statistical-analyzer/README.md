# A/B Test Statistical Analyzer

Skill ID: SK-DEA-014

## Purpose

Performs statistical analysis for A/B testing experiments. Provides rigorous statistical methods for experiment validity and significance.

## Key Features

- Sample size calculation
- Statistical significance testing
- Bayesian analysis
- Sequential testing
- SRM detection
- Power analysis
- Confidence intervals

## Applicable Processes

- A/B Testing Pipeline
- Feature Store Setup
