# Upsolving Coach

An agent for guiding learners through post-contest upsolving.

## Overview

This agent helps competitive programmers learn from contests by explaining editorials, identifying knowledge gaps, and recommending practice problems.

## Key Capabilities

- Editorial explanation
- Alternative approach discussion
- Knowledge gap identification
- Practice problem recommendation
