# AtCoder Client

A skill for interfacing with the AtCoder competitive programming platform.

## Overview

This skill provides access to AtCoder contests, problems, and user statistics. AtCoder is a Japanese competitive programming platform known for high-quality algorithmic problems.

## Key Features

- Contest problem fetching with English translations
- Problem difficulty ratings via AtCoder Problems
- Submission tracking and verdict retrieval
- Virtual contest management
- User statistics and progress tracking

## Platform Reference

- AtCoder: https://atcoder.jp
- AtCoder Problems: https://kenkoooo.com/atcoder/
