# Suffix Structure Builder

A skill for building suffix arrays and related structures.

## Overview

This skill provides implementations for suffix-based data structures with efficient construction algorithms and query support.

## Key Features

- Suffix array construction
- LCP array building
- Suffix tree implementation
- Suffix automaton generation
- Query method implementations

## Structures Supported

- Suffix Array (SA-IS, DC3)
- LCP Array (Kasai's algorithm)
- Suffix Tree (Ukkonen's)
- Suffix Automaton
