# CSES Tracker

A skill for tracking progress through the CSES Problem Set.

## Overview

The CSES Problem Set is a comprehensive collection of competitive programming problems designed for systematic skill building. This skill helps track progress, suggest next problems, and map learning to the Competitive Programmer's Handbook.

## Key Features

- Progress tracking by category
- Difficulty-based problem suggestions
- CP Handbook chapter mapping
- Gap analysis and learning paths
- Progress report generation

## Reference

- CSES Problem Set: https://cses.fi/problemset/
- Competitive Programmer's Handbook: https://cses.fi/book/
