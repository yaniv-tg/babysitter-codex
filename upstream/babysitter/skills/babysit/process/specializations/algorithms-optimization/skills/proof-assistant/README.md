# Proof Assistant

A skill for constructing algorithm correctness proofs.

## Overview

This skill assists in constructing formal correctness proofs for algorithms using standard mathematical proof techniques.

## Key Features

- Proof structure templates
- Step-by-step guidance
- Termination arguments
- Proof validation
- Gap identification

## Proof Techniques

- Mathematical Induction
- Proof by Contradiction
- Loop Invariant Proofs
- Structural Induction
