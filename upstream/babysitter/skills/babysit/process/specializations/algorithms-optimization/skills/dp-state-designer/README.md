# DP State Designer

A skill for designing optimal dynamic programming solutions.

## Overview

This skill helps design dynamic programming solutions by identifying optimal state representations, deriving transition formulas, and suggesting space/time optimizations.

## Key Features

- Subproblem structure identification
- State parameter selection
- Transition formula derivation
- Complexity estimation
- Space optimization suggestions

## DP Design Process

1. Identify subproblems
2. Define state parameters
3. Derive transitions
4. Establish base cases
5. Determine computation order
6. Apply optimizations
