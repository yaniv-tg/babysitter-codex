# Flow Network Builder

A skill for modeling problems as network flow.

## Overview

This skill helps identify when problems can be modeled as network flow and constructs the appropriate flow network with source, sink, and edge capacities.

## Key Features

- Max-flow problem modeling
- Min-cut identification
- Bipartite matching reduction
- Min-cost flow construction
- Algorithm selection

## Flow Problem Types

- Maximum Flow
- Minimum Cut
- Bipartite Matching
- Min-Cost Max-Flow
- Circulation
