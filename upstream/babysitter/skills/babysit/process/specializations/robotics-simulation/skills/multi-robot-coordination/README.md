# Multi-Robot Coordination Skill

Coordination and task allocation for multi-robot systems and fleets.

## Purpose

Provides expert capabilities for coordinating multiple robots, including task allocation, multi-agent path planning, and fleet-level collision avoidance.

## Key Features

- Task allocation algorithms
- Conflict-based path planning
- ORCA/RVO collision avoidance
- Formation control
- Fleet management integration
- Traffic zone management

## When to Use

- Coordinating robot fleets
- Implementing multi-robot path planning
- Setting up task allocation systems
- Configuring collision avoidance
- Building fleet management systems
