# Behavior Trees Skill

Behavior tree design and implementation for robot decision making.

## Purpose

Provides expert capabilities for designing behavior trees that control robot decision-making, enabling modular, reactive, and maintainable robot behaviors.

## Key Features

- BehaviorTree.CPP configuration
- Custom node implementation
- Blackboard state management
- Groot debugging integration
- Reactive behavior design
- Recovery behavior patterns

## When to Use

- Designing robot task logic
- Implementing navigation behaviors
- Building reactive decision systems
- Creating modular behavior architectures
- Debugging behavior execution
