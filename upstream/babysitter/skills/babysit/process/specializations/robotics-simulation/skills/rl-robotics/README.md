# Reinforcement Learning Skill

RL training for robot control using simulation with sim-to-real transfer.

## Purpose

Provides expert capabilities for training reinforcement learning policies for robot control, including simulation environment design, training optimization, and deployment preparation.

## Key Features

- Gymnasium environment creation
- Stable Baselines3 training
- Reward shaping strategies
- Domain randomization
- Curriculum learning
- Policy export for deployment

## When to Use

- Training RL policies for robot control
- Designing simulation environments
- Implementing sim-to-real transfer
- Optimizing training for robotics
- Deploying learned policies
