# Nav2 Navigation Skill

Specialized skill for ROS2 Nav2 navigation stack configuration and behavior trees.

## Purpose

Provides expert capabilities for configuring the Nav2 navigation stack for autonomous mobile robots, including path planning, obstacle avoidance, and behavior-based navigation control.

## Key Features

- Nav2 stack configuration
- Behavior tree design and customization
- Costmap layer management
- Planner and controller selection
- Recovery behavior implementation
- Multi-floor navigation support

## When to Use

- Setting up mobile robot navigation
- Configuring costmaps for obstacle avoidance
- Designing complex navigation behaviors
- Implementing waypoint following
- Debugging navigation failures
