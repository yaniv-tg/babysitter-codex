# NVIDIA Isaac Sim Skill

Specialized skill for NVIDIA Isaac Sim photorealistic simulation and synthetic data generation.

## Purpose

Provides expert capabilities for creating photorealistic robot simulation environments using NVIDIA Isaac Sim, generating synthetic training data with domain randomization, and configuring high-fidelity physics simulation.

## Key Features

- USD scene creation and management
- PhysX physics configuration
- Replicator synthetic data pipelines
- Domain randomization strategies
- ROS/ROS2 bridge integration
- Multi-GPU simulation setup

## When to Use

- Creating photorealistic simulation environments
- Generating synthetic training datasets
- Implementing domain randomization for sim-to-real
- Developing digital twins with visual fidelity
- Training vision-based robot learning systems
