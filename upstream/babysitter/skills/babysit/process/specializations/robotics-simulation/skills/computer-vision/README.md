# Computer Vision Skill

Specialized skill for robot vision including feature detection, tracking, and camera calibration.

## Purpose

Provides comprehensive capabilities for implementing computer vision in robotics, including camera calibration, feature extraction, visual tracking, and image-based robot control.

## Key Features

- Camera intrinsic/extrinsic calibration
- Stereo vision and depth estimation
- Feature detection and matching
- Optical flow tracking
- Marker detection (ArUco, AprilTag)
- Visual servoing

## When to Use

- Calibrating robot cameras
- Implementing visual tracking systems
- Setting up stereo vision pipelines
- Detecting fiducial markers
- Building visual servoing controllers
