## Babysitter SDK Documentation

### Core SDK
- **SDK Repository**: Internal Babysitter SDK documentation
- **Process Patterns**: `plugins/babysitter/skills/babysit/process/` - Example processes
- **Methodology Library**: `plugins/babysitter/skills/babysit/process/methodologies/`

### Process Definition Patterns
- **defineTask API**: Task definition with kind (agent, skill, node, shell, breakpoint)
- **ctx.task()**: Execute tasks within process flow
- **ctx.breakpoint()**: Human-in-the-loop approval gates
- **ctx.log()**: Structured logging for observability
