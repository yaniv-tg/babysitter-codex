# Stakeholder Communication Skill

Generate stakeholder-specific communications and presentations for product updates.

## Purpose

This skill provides comprehensive stakeholder communication capabilities. It helps product teams create targeted communications that resonate with different audiences, from executives to customers to internal teams.

## Key Capabilities

- **Executive Communications**: Summaries, board updates, investor communications
- **Sales Enablement**: Feature briefs, competitive positioning, talk tracks
- **Customer Communications**: Announcements, release notes, success updates
- **Internal Updates**: Launch announcements, FAQs, status reports
- **Presentation Support**: Slide outlines, talking points, Q&A prep

## Process Integration

Integrates with:
- Stakeholder Alignment Process
- Product Council Review Process
- Product Launch GTM Process
- Quarterly Roadmap Process

## Usage

Invoke this skill when preparing communications for different stakeholder groups, creating presentations, or generating FAQ documents.
