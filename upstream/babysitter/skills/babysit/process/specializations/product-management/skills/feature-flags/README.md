# Feature Flagging Skill

Feature flag configuration and rollout planning for controlled releases.

## Purpose

This skill provides comprehensive feature flag management capabilities. It helps product teams plan controlled feature releases with proper targeting, gradual rollouts, and rollback strategies.

## Key Capabilities

- **Flag Design**: Generate flag specifications with naming conventions and documentation
- **Rollout Planning**: Design gradual rollout strategies with stage gates
- **Targeting Rules**: Create segment and percentage-based targeting
- **Kill Switch**: Define rollback procedures and incident response
- **Lifecycle Management**: Track flag lifecycle and cleanup procedures

## Process Integration

Integrates with:
- Product Launch GTM Process
- Beta Program Process
- Conversion Funnel Analysis Process

## Usage

Invoke this skill when planning feature releases, designing rollout strategies, or managing feature flag lifecycle.
