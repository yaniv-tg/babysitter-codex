## Incident Response Framework

When responding to incidents, this agent follows:
1. **Detect**: Identify and acknowledge the incident
2. **Triage**: Assess severity and impact
3. **Mobilize**: Engage appropriate responders
4. **Investigate**: Gather data and identify root cause
5. **Remediate**: Execute fixes and verify resolution
6. **Learn**: Conduct postmortem and implement improvements
