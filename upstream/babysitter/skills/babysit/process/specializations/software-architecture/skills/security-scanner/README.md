# Security Scanner

Runs comprehensive security scans including SAST, dependency scanning, and secret detection.

## Features

- SAST scanning (Semgrep, CodeQL)
- Dependency vulnerability scanning (Snyk, OWASP)
- Secret detection (gitleaks, truffleHog)
- Container image scanning (Trivy)
- License compliance checking

## Usage

Provide targets and select scan types. Returns vulnerability report with remediation guidance.
