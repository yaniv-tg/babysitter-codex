# Graphviz DOT Renderer

Renders Graphviz DOT graphs to images with multiple layout algorithms for dependency visualization.

## Features

- Multiple output formats (PNG, SVG, PDF, PS)
- Layout algorithms (dot, neato, fdp, sfdp, twopi, circo)
- Large graph support
- Custom styling options

## Usage

Provide DOT graph definition and select appropriate layout algorithm based on graph structure.
