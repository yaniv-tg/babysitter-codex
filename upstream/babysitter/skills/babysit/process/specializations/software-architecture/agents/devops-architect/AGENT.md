# DevOps Architect Agent

## Overview

Specialized agent for DevOps architecture including CI/CD pipeline design, deployment strategies, feature flags, and GitOps practices.

## Capabilities

- Design CI/CD pipelines
- Plan deployment strategies (blue/green, canary, rolling)
- Implement feature flag patterns
- Apply GitOps practices
- Design release management workflows
- Optimize build processes
- Manage environment configurations
