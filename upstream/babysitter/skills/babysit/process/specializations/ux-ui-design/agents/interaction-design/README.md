# Interaction Design Agent

Define and validate interaction patterns and micro-interactions.

## Overview

This agent creates engaging interaction designs with proper timing, easing, and feedback patterns that enhance user experience.

## Key Capabilities

- **Micro-Interactions**: Trigger, feedback, loop design
- **Motion Specification**: Timing and easing curves
- **State Transitions**: Smooth state change animations
- **Accessibility**: Reduced motion alternatives

## When to Use

- Designing component interactions
- Creating motion design guidelines
- Specifying animation behavior
- Building interaction documentation
