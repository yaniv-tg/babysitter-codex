# Storybook Integration Agent

Manage Storybook component documentation and visual testing.

## Overview

This agent configures Storybook for comprehensive component documentation, including interactive examples, controls, and visual regression testing.

## Key Capabilities

- **Story Generation**: CSF 3.0 format stories
- **Props Documentation**: Automatic extraction
- **Visual Regression**: Test setup and configuration
- **Addon Setup**: Controls, a11y, viewport

## When to Use

- Setting up component documentation
- Creating interactive examples
- Configuring visual regression tests
- Building component playgrounds
