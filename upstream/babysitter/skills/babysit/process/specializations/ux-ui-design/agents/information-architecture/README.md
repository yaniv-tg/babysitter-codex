# Information Architecture Agent

Design and validate information architecture for digital products.

## Overview

This agent creates and validates information architecture, ensuring content is organized logically and users can navigate efficiently.

## Key Capabilities

- **Site Map Generation**: Hierarchical structure creation
- **Navigation Design**: Primary/secondary navigation
- **Labeling Systems**: Consistent terminology
- **Findability Testing**: Content discovery analysis

## When to Use

- Starting new website/app design
- Restructuring existing content
- Validating navigation effectiveness
- Card sorting analysis
