# User Research Synthesizer Agent

Analyze and synthesize user research data into actionable insights.

## Overview

This agent processes qualitative and quantitative user research data, identifying patterns and extracting insights to inform design decisions.

## Key Capabilities

- **Thematic Analysis**: Code and categorize qualitative data
- **Statistical Analysis**: Process survey and quantitative data
- **Pattern Recognition**: Identify behavioral patterns
- **Insight Generation**: Extract actionable findings

## When to Use

- After completing user interviews
- Analyzing survey results
- Synthesizing multiple research studies
- Creating research reports
