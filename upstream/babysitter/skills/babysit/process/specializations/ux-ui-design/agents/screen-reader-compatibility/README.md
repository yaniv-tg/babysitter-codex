# Screen Reader Compatibility Agent

Validate screen reader experiences across assistive technologies.

## Overview

This agent specializes in testing and validating how content is experienced through screen readers, ensuring accessibility for users with visual impairments.

## Key Capabilities

- **Multi-AT Testing**: NVDA, JAWS, VoiceOver, TalkBack
- **ARIA Validation**: Role, state, and property verification
- **Focus Testing**: Navigation and reading order
- **Test Script Generation**: Manual testing documentation

## When to Use

- Validating form accessibility
- Testing dynamic content announcements
- Verifying modal and dialog behavior
- Checking navigation landmarks
