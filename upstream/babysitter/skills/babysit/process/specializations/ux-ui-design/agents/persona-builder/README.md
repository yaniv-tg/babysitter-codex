# Persona Builder Agent

Generate research-backed user personas from user research data.

## Overview

This agent creates comprehensive user personas that represent target user segments, grounded in research data and actionable for design decisions.

## Key Capabilities

- **User Segmentation**: Identify distinct user groups
- **Profile Development**: Demographics and psychographics
- **Empathy Mapping**: Says/thinks/does/feels analysis
- **Research Validation**: Ensure personas are data-backed

## When to Use

- After completing user research
- Starting a new product design
- Refreshing outdated personas
- Aligning team on target users
