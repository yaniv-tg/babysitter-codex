# Browser Stack Skill

Cross-browser and cross-device testing using cloud testing platforms.

## Overview

This skill integrates with BrowserStack and Sauce Labs to perform comprehensive cross-browser and cross-device testing on real devices and browsers.

## Key Features

- **Real Devices**: Test on actual mobile devices
- **Browser Coverage**: All major browsers and versions
- **Screenshot Capture**: Visual documentation
- **Compatibility Reports**: Detailed compatibility matrices
- **Visual Regression**: Compare across platforms

## When to Use

- Validating responsive designs across devices
- Testing browser compatibility before release
- Debugging device-specific issues
- Creating compatibility documentation

## Supported Platforms

| Platform | Browsers | Devices |
|----------|----------|---------|
| BrowserStack | Chrome, Firefox, Safari, Edge | iOS, Android devices |
| Sauce Labs | Chrome, Firefox, Safari, Edge | iOS, Android devices |
