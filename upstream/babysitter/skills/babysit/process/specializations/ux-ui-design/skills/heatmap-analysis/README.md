# Heatmap Analysis Skill

Analyze user interaction heatmaps for attention patterns and behavioral insights.

## Overview

This skill processes heatmap data from various analytics platforms to identify user engagement patterns and generate actionable UX recommendations.

## Key Features

- **Hotspot Detection**: Identify high-engagement areas
- **Cold Zone Analysis**: Find overlooked content areas
- **Scroll Depth Tracking**: Measure content consumption
- **Click Pattern Analysis**: Understand interaction behavior
- **Comparative Analysis**: Compare heatmaps across variants

## When to Use

- Analyzing landing page effectiveness
- Identifying content placement issues
- Validating navigation patterns
- A/B test visual analysis
- Understanding mobile vs desktop behavior

## Data Sources

| Platform | Data Type | API Support |
|----------|-----------|-------------|
| Hotjar | Click, move, scroll | Yes |
| Crazy Egg | Click, scroll, attention | Yes |
| Clarity | Click, scroll | Yes |
| Custom | JSON format | Manual |
