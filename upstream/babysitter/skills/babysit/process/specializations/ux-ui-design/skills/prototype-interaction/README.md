# Prototype Interaction Skill

Define and document prototype interactions for design handoff.

## Overview

This skill helps define interactive prototype behaviors, including hotspots, transitions, and navigation flows, generating specifications for designers and developers.

## Key Features

- **Hotspot Mapping**: Define clickable/tappable areas
- **Transition Definitions**: Specify animations and timing
- **Interaction Matrix**: Document all screen connections
- **Multi-Format Export**: JSON, Figma, InVision, Markdown
- **Gesture Support**: Tap, click, hover, drag, swipe

## When to Use

- Creating interactive prototype specifications
- Documenting interaction designs for handoff
- Defining micro-interactions and transitions
- Building click-through prototypes

## Transition Types

| Type | Description | Duration |
|------|-------------|----------|
| slide-left | Enter from right | 200-400ms |
| slide-up | Enter from bottom | 200-400ms |
| fade | Opacity transition | 150-300ms |
| scale | Zoom in/out | 200-300ms |
| dissolve | Cross-fade | 200-400ms |
