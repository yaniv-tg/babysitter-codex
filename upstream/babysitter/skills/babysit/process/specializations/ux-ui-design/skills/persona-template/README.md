# Persona Template Skill

Generate user persona documents and empathy maps from templates.

## Overview

This skill transforms persona research data into professional documents and visualizations, streamlining the persona documentation process.

## Key Features

- **Template Library**: Multiple persona card styles
- **Empathy Maps**: Visual empathy map generation
- **Multi-Format Export**: PDF, PNG, Markdown, HTML
- **Customization**: Flexible template styling
- **Comparison Views**: Side-by-side personas

## When to Use

- Documenting user research findings
- Creating stakeholder presentations
- Building persona libraries
- Team alignment workshops

## Template Styles

| Style | Best For |
|-------|----------|
| Minimal | Quick reference cards |
| Detailed | Comprehensive documentation |
| Visual | Presentations |
| UX Agency | Client deliverables |

## Persona Elements

- Demographics and photo
- Goals and motivations
- Frustrations and pain points
- Behaviors and habits
- Representative quote
- Biographical narrative
