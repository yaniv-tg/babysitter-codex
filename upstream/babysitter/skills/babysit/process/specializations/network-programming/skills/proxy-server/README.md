# Proxy Server Skill

Expert skill for proxy server implementation, configuration, and traffic interception.

## Overview

This skill provides specialized capabilities for building and configuring proxy servers including HTTP, SOCKS, and transparent proxies. It covers traffic interception, caching, and PAC file generation.

## Key Features

- Squid, mitmproxy, nginx proxy configuration
- HTTP CONNECT tunneling implementation
- SOCKS4/SOCKS5 proxy support
- Transparent proxying with iptables
- PAC file generation

## When to Use

- Setting up forward or reverse proxies
- Implementing traffic interception
- Building SOCKS proxies
- Configuring transparent proxying
- Debugging HTTP traffic

## Dependencies

- Squid
- mitmproxy
- nginx
- iptables/nftables
