# OpenAPI Skill

OpenAPI specification for API documentation and code generation.

## Overview

This skill provides expertise in OpenAPI (formerly Swagger) for documenting and validating REST APIs.

## When to Use

- Documenting APIs
- Generating client SDKs
- Contract-first development
- API testing

## Quick Start

```yaml
openapi: 3.0.3
info:
  title: My API
  version: 1.0.0
paths:
  /users:
    get:
      responses:
        '200':
          description: OK
```

## Integration

Works with rest-api-design-skill for API planning.
