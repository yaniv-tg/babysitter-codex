# REST API Design Skill

RESTful API design principles and best practices for building scalable APIs.

## Overview

This skill provides expertise in REST API design, covering resource modeling, HTTP semantics, and API conventions.

## When to Use

- Designing new APIs
- Reviewing API architecture
- Implementing pagination
- Defining error handling

## Key Principles

| Principle | Description |
|-----------|-------------|
| Resources | Nouns, not verbs |
| HTTP Methods | GET, POST, PUT, DELETE |
| Status Codes | Semantic responses |
| Versioning | /v1/users or Accept header |

## Integration

Pairs with openapi-skill for documentation.
