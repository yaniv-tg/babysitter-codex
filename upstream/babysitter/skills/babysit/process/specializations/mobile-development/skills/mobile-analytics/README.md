# Mobile Analytics Skill

## Overview

The Mobile Analytics skill provides comprehensive capabilities for mobile app analytics and crash reporting integration. It enables configuration of analytics platforms, event tracking, user segmentation, and A/B testing.

## Key Capabilities

- **Firebase Analytics**: Events, user properties, audiences
- **Crashlytics**: Crash reporting, custom keys, alerting
- **Third-Party**: Mixpanel, Amplitude, Segment
- **A/B Testing**: Remote Config, experiments
- **Funnels**: Conversion tracking, retention

## Quick Start

### Firebase Setup

```bash
# Install Firebase CLI
npm install -g firebase-tools

# Initialize Firebase
firebase init
```
