# LLVM Backend Skill

Expert skill for LLVM integration including IR generation, optimization passes, and native code emission.

## Quick Reference

- **Category**: Code Generation
- **ID**: SK-005

## Key Capabilities

- LLVM IR generation from high-level AST/IR
- LLVM optimization pass configuration
- Custom LLVM pass implementation
- LLVM type system mapping
- DWARF debug information generation
- Target machine and codegen configuration
- LLVM JIT integration (ORC, MCJIT)
- Cross-compilation support

## When to Use

Use this skill when implementing:
- Native code generation backends
- JIT compilation using LLVM
- Debug information generation
- Custom optimization passes
- Cross-compilation support
