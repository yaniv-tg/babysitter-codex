# Error Messages Skill

Expert skill for designing and implementing high-quality compiler error messages.

## Quick Reference

- **Category**: User Experience
- **ID**: SK-014

## Key Capabilities

- Clear, actionable error message templates
- Source context display with underlines
- Fix suggestions and quick fixes
- Error cascade handling and suppression
- Multi-span error annotations
- Machine-readable error output (JSON, SARIF)
- Color/styled terminal output
- Error recovery strategies for better diagnostics

## When to Use

Use this skill when implementing:
- Compiler error reporting
- Type error diagnostics
- Parser error recovery
- IDE error display
- CI/CD error formatting
