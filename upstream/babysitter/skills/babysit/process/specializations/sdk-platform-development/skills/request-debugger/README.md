# Request Debugger

HTTP request/response debugging and inspection tools.

## Purpose

This skill implements debugging tools that help developers understand and troubleshoot HTTP interactions within SDKs.

## Key Features

- Request/response logging
- Sensitive data redaction
- Curl command export
- Timing breakdowns
- HAR export

## When to Use

- Debugging API issues
- Reproducing requests
- Analyzing performance
- Troubleshooting failures
