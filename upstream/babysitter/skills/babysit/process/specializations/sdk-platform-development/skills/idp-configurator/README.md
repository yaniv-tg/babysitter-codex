# IDP Configurator

Configure Internal Developer Platform (IDP) components.

## Purpose

This skill configures internal developer platforms including service catalogs, templates, and self-service workflows for development teams.

## Key Features

- Service catalog setup
- Golden path templates
- Self-service workflows
- CI/CD integration
- TechDocs integration

## When to Use

- Setting up Backstage
- Creating software templates
- Implementing self-service
- Building internal portals
