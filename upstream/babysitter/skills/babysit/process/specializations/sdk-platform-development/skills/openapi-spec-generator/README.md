# OpenAPI Spec Generator

Automated OpenAPI specification generation from code annotations, comments, and interface definitions.

## Purpose

This skill parses source code to automatically generate and maintain OpenAPI 3.x specifications, ensuring API documentation stays synchronized with implementation.

## Key Features

- Multi-framework support (Express, FastAPI, Spring, ASP.NET)
- Automatic schema extraction from type definitions
- Incremental spec updates from code changes
- Built-in validation against OpenAPI standards

## When to Use

- Setting up API documentation from existing code
- Maintaining API specs as code-first approach
- Validating API implementation matches design
- Generating SDK input specifications
