# Performance Benchmark Suite

SDK performance benchmarking and regression detection.

## Purpose

This skill implements comprehensive performance benchmarking for SDKs, enabling detection of regressions and tracking performance over time.

## Key Features

- Latency percentile tracking
- Memory profiling
- Regression detection
- Visual reports
- CI integration

## When to Use

- Establishing performance baselines
- Detecting performance regressions
- Comparing SDK versions
- Validating performance SLOs
