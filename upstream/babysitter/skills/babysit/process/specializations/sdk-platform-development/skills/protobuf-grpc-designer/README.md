# Protocol Buffers and gRPC Designer

Protocol Buffers and gRPC service definition with backward compatibility checks.

## Purpose

This skill designs efficient, backward-compatible Protocol Buffers schemas and gRPC service definitions for high-performance cross-language APIs.

## Key Features

- Google style guide compliant .proto files
- Backward compatibility validation with buf
- Multi-language stub generation
- Streaming pattern support (all types)
- gRPC-Gateway REST transcoding

## When to Use

- Designing high-performance internal APIs
- Building microservice communication layers
- Creating cross-language service interfaces
- Implementing real-time streaming features
