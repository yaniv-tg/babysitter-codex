# Developer Portal Builder

Build unified developer portals with Backstage or custom frameworks.

## Purpose

This skill creates comprehensive developer portals that provide a unified experience for API discovery, documentation, credentials, and analytics.

## Key Features

- API catalog and discovery
- Credential management
- Usage analytics dashboards
- Self-service onboarding
- Multi-tenant support

## When to Use

- Building public developer portals
- Creating internal developer platforms
- Unifying API management UIs
- Improving developer self-service
