# Go SDK Specialist

Go SDK development with idiomatic patterns.

## Purpose

This skill develops production-ready Go SDKs that follow Go idioms, use context-based patterns, and implement proper error handling.

## Key Features

- Idiomatic Go API design
- Context-based cancellation
- Functional options pattern
- Proper error wrapping
- Go modules support

## When to Use

- Building Go client libraries
- Implementing concurrent API clients
- Creating testable Go SDKs
- Publishing Go modules
