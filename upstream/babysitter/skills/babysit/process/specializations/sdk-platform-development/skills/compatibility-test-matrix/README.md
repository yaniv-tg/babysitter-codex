# Compatibility Test Matrix

Multi-version, multi-platform SDK compatibility testing.

## Purpose

This skill implements comprehensive compatibility testing to ensure SDKs work reliably across all supported environments and versions.

## Key Features

- Multi-runtime testing
- Cross-OS validation
- Dependency compatibility
- Browser testing
- CI matrix automation

## When to Use

- Validating SDK compatibility
- Setting up CI matrices
- Documenting support policy
- Testing dependency updates
