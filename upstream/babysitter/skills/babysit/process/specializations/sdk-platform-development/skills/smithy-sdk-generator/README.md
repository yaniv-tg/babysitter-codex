# Smithy SDK Generator

AWS Smithy-based SDK generation for enterprise-grade APIs.

## Purpose

This skill uses AWS Smithy to generate production-ready SDKs with enterprise features like waiters, paginators, and automatic retries built-in.

## Key Features

- Protocol-agnostic API modeling
- AWS-style SDK patterns
- Multi-language SDK generation
- Built-in waiters and paginators
- Custom trait and plugin support

## When to Use

- Building enterprise-grade API platforms
- Creating AWS-style SDK experiences
- Generating SDKs with advanced features
- Implementing protocol-agnostic APIs
