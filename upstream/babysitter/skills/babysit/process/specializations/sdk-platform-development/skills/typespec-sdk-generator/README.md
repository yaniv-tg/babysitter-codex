# TypeSpec SDK Generator

Microsoft TypeSpec-based API and SDK generation.

## Purpose

This skill uses Microsoft TypeSpec to design APIs with a concise, type-safe language and generate production-ready SDKs following Azure SDK guidelines.

## Key Features

- Concise TypeSpec API definitions
- Multi-language SDK generation
- Azure SDK guidelines compliance
- OpenAPI emission
- Versioning and deprecation support

## When to Use

- Designing APIs with TypeSpec language
- Generating Azure-style SDKs
- Creating consistent multi-language clients
- Migrating from OpenAPI to TypeSpec
