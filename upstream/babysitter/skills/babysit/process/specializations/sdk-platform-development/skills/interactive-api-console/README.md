# Interactive API Console

Build interactive API try-it-out consoles for documentation.

## Purpose

This skill creates embedded API explorers that allow developers to test API calls directly from documentation, improving time-to-first-call.

## Key Features

- Embedded API exploration
- Multi-environment support
- Code snippet generation
- Authentication integration
- Request/response visualization

## When to Use

- Adding try-it-out to documentation
- Building API playgrounds
- Improving developer onboarding
- Creating interactive tutorials
