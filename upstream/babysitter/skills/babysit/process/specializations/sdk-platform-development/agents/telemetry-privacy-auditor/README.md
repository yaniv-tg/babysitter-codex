# Telemetry Privacy Auditor Agent

Audits SDK telemetry for privacy compliance.

## Purpose

This agent ensures SDK telemetry implementations comply with privacy regulations and respect user privacy.

## Key Responsibilities

- Reviewing data collection
- Verifying anonymization
- Checking consent mechanisms
- Ensuring regulatory compliance

## When to Engage

- Auditing telemetry privacy
- Checking GDPR compliance
- Reviewing data collection
- Assessing anonymization
