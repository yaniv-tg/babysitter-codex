# Unreal Networking Skill

## Overview

The Unreal Networking skill enables multiplayer game development using Unreal's built-in networking.

## Purpose

Unreal provides robust networking. This skill enables:

- **Replication**: Property synchronization
- **RPCs**: Remote function calls
- **Authority**: Server-authoritative design
- **Dedicated Servers**: Scalable hosting

## Use Cases

1. **Competitive Games**: Authoritative servers
2. **Co-op Games**: Session-based play
3. **MMO Lite**: Persistent worlds
4. **LAN Games**: Local multiplayer

## Processes That Use This Skill

- **Multiplayer Networking** (`multiplayer-networking.js`)

## Related Skills

- `unreal-development`: Core Unreal development
- `client-server`: Architecture patterns
- `lag-compensation`: Prediction systems
