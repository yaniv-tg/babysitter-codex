# Unity Animation Skill

## Overview

The Unity Animation skill enables character and object animation using Animator controllers, Animation Rigging, and Timeline.

## Purpose

Animation brings games to life. This skill provides:

- **State Machines**: Animator controllers
- **Blending**: Blend trees and layers
- **Procedural**: Animation Rigging IK
- **Cinematic**: Timeline sequences

## Use Cases

1. **Character Animation**: Movement, combat, interactions
2. **Cutscenes**: Scripted sequences
3. **Procedural Animation**: Look-at, IK
4. **UI Animation**: Animated interfaces

## Processes That Use This Skill

- **Animation Implementation** (`animation-implementation.js`)
- **Cutscene Implementation** (`cutscene-implementation.js`)

## Related Skills

- `unity-development`: Core Unity development
- `animation-blending`: Blend techniques
- `animation-state-machines`: FSM patterns
