# Godot GDScript Skill

## Overview

The Godot GDScript skill enables game development using Godot's native scripting language.

## Purpose

GDScript provides tight engine integration. This skill enables:

- **Rapid Development**: Python-like syntax
- **Engine Integration**: Direct node access
- **Signals**: Event-driven programming
- **Performance**: Typed GDScript optimization

## Use Cases

1. **Gameplay Logic**: Player, enemies, mechanics
2. **UI Systems**: Menu, HUD, dialogs
3. **Level Scripting**: Triggers, events
4. **Prototyping**: Quick iterations

## Processes That Use This Skill

- **Core Mechanics Prototyping** (`core-mechanics-prototyping.js`)
- **Gameplay Systems** (`gameplay-systems.js`)

## Related Skills

- `godot-development`: Core Godot development
- `godot-csharp`: C# alternative
- `fsm`: State machine patterns
