# Unreal MetaSounds Skill

## Overview

The Unreal MetaSounds skill enables procedural audio creation in Unreal Engine 5.

## Purpose

MetaSounds provides programmable audio. This skill enables:

- **Synthesis**: Generate sounds in real-time
- **Procedural**: Variation and adaptation
- **Node-Based**: Visual audio programming
- **Integration**: Gameplay-driven audio

## Use Cases

1. **Weapon Sounds**: Procedural variations
2. **Vehicles**: Engine synthesis
3. **Ambient**: Dynamic environments
4. **Music**: Adaptive scores

## Processes That Use This Skill

- **Audio Integration** (`audio-integration.js`)

## Related Skills

- `unreal-development`: Core Unreal development
- `procedural-audio`: Synthesis concepts
- `adaptive-music`: Interactive music
