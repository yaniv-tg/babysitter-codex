# Godot C# Skill

## Overview

The Godot C# skill enables game development using C# and .NET in Godot Engine.

## Purpose

C# provides industry-standard programming. This skill enables:

- **.NET Integration**: Full .NET ecosystem
- **Type Safety**: Strong typing
- **Tooling**: IDE support, debugging
- **Libraries**: NuGet packages

## Use Cases

1. **Complex Systems**: Large codebases
2. **Code Reuse**: Shared .NET libraries
3. **Enterprise**: Team development
4. **Migration**: Unity developers

## Processes That Use This Skill

- **Gameplay Systems** (`gameplay-systems.js`)
- **Performance Optimization** (`performance-optimization.js`)
