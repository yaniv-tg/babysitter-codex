# Unity UI Toolkit Skill

## Overview

The Unity UI Toolkit skill enables modern UI development using UXML templates, USS styling, and custom visual elements.

## Purpose

UI Toolkit is Unity's modern UI system. This skill provides:

- **UXML Templates**: XML-based UI structure
- **USS Styling**: CSS-like styling system
- **Visual Elements**: Composable UI components
- **Data Binding**: Reactive UI patterns

## Use Cases

1. **Game Menus**: Main menu, settings, inventory
2. **HUD Elements**: Health bars, minimaps, indicators
3. **Editor Extensions**: Custom Unity editor tools
4. **Complex Layouts**: Dynamic, responsive UIs

## Processes That Use This Skill

- **UI/UX Implementation** (`uiux-implementation.js`)
- **Menu Systems** (`menu-systems.js`)
- **Localization Process** (`localization-process.js`)
