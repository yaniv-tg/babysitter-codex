# Blessed Widget Generator Skill

Generate blessed widgets for Node.js terminal dashboards and forms.

## Overview

This skill generates terminal UI widgets using blessed and blessed-contrib, enabling the creation of complex dashboards, forms, and interactive interfaces in Node.js.

## When to Use

- Building terminal dashboards
- Creating monitoring interfaces
- Implementing complex TUI layouts
- Setting up blessed projects

## Quick Start

```json
{
  "projectName": "my-dashboard",
  "layout": "dashboard",
  "widgets": ["line-chart", "gauge", "table", "log"]
}
```

## Features

- Dashboard layouts with grid
- Charts and gauges
- Interactive tables
- Form widgets
- Event handling

## Integration with Processes

| Process | Integration |
|---------|-------------|
| tui-application-framework | Blessed setup |
| dashboard-monitoring-tui | Dashboard widgets |
| interactive-form-implementation | Form components |
