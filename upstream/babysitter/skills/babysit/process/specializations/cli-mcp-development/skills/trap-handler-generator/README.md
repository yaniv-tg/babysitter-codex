# Trap Handler Generator Skill

Generate trap handlers for cleanup and signal handling in shell scripts.

## Overview

This skill generates trap handlers for shell scripts, implementing proper cleanup, signal handling, and graceful shutdown patterns.

## Features

- Cleanup trap handlers
- Signal handling (INT, TERM)
- Error handlers with line numbers
- Resource registration
- Nested trap support
